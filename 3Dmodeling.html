The main use of Blender; 3D modeling

3D-modeling, alongside 3D animating, is the main function of Blender. Despite them both being the main functions of Blender, you must first learn at least the basics of 3D-modeling before you can proceed to animating.

What IS 3D modeling, exactly?

The Wikipedia definition of 3D modeling is as follows:

"3D modeling is the process of developing a mathematical coordinate-based representation of any surface of an object (inanimate or living) in three dimensions via specialized software."

Sounds complicated, does it not? Though it may sound complicated, in reality, 3D modeling software has come a long way and it's been a long time since you last needed to input mathematical calculations to define an object.

The user has access to an array of tools, many of which are separated depending on how they interact with objects and are available in different Object Modes. The Object Modes of Blender are:

  • Object Mode - allows you to see objects whole with filters which you applied to them and interact with them in a limited way.
  • Edit Mode - allows you to alter the shape of the object by manipulating its vertexes, edges and faces.
  • Sculpt Mode - allows you to alter the shape of an object through the use of the Brush; rather than manipulating individual entities mentioned above, the Brush manipulates the geometry of an area.
  • Vertex Paint Mode - allows you to color vertexes.
  • Weight Paint Mode - allows you to "paint weight" onto an area. You paint the area with the Brush in order to add weight; keep brushing over one area in order to add more weight.
  • Texture Paint Mode - a mesh-only mode which allows you to paint your mesh's texture directly on the model.
  • Particle Edit Mode - a mesh-only mode useful for configurable particle systems, used to create realistic hair.
  • Pose Mode - an armature only mode dedicated to armature posing.
  • Paint Mode - a Grease Pencil only mode, dedicated to Grease Pencil strokes.

Modeling made easy

Using different Modes you have the perfect tools to create just the model you wanted; Blender is used commonly both hobbystically and profesionally. How you're gonna use Blender - is up to you!
